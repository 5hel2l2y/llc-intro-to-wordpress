##Getting started Create an account at [wordpress.com](http://wordpress.com) and follow the setup instructions. When creating/using a WordPress.com account, your domain name will have `.wordpress.com` appended to it. ![](framework/img/workshop/getting-started-create-site.png) If you already have an account you'd like to use, then there's no need to do any setup. Just come to the workshop with your login info! **Important**: If you have an existing account and want to create another blog, WordPress.com will not let you create multiple accounts with the same email or username. (Though you can add users and update the username display - more on this in the workshop.)
##Getting started You have the option to register a custom domain with Wordpress in this step. It's optional and can also be added at a later date. You can also choose to register with other domain registration companies such as [hover.com](http://hover.com/ladiescode). More on this in the workshop. ![](framework/img/workshop/getting-started-address.png)
##Getting started Pick a plan. Try the free plan just to test it out first. You can always upgrade to a different plan later, or learn how to self-host and customize it yourself! Read more about the [different plans here](https://learn.wordpress.com/get-started/#wpplan). ![](framework/img/workshop/getting-started-plan.png)
